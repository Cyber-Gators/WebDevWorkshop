CSS Box Model, Width & Height, Overflow, and Box Sizing

Content Box (box-sizing: content-box)

This box uses the default box-sizing: content-box, where the width and height apply only to the content area. Padding and borders are added to the total size.

Border Box (box-sizing: border-box)

This box uses box-sizing: border-box, which includes the padding and border within the specified width and height.

Overflow Box (Overflow: auto)

This box has a fixed width and height with overflow set to auto. If the content exceeds the box size, a scrollbar will appear.
